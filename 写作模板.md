# 文章标题写在这里

> 散文 | 乙巳年六月初一

正文从这一行开始写，段落之间空一行。

如果想分小节，可以像这样写小节标题（行首两个 # 号）：

## 第一节

正文照常写。需要加粗的地方，用两个星号包住：**这里就是粗体**。

如果想在文章里插入照片：
1. 把照片文件（例如 西湖.jpg）放进网站文件夹的 images 文件夹
2. 在正文单独写一行：![西湖春日](images/西湖.jpg)
（也支持直接写 HTML 标签：<img src="images/西湖.jpg" alt="西湖春日" class="dx-image">，两种写法效果相同；
注意图片文件名要与引用完全一致，包括扩展名 .jpg / .webp 等）

写完后：
1. 把本文件另存到 articles 文件夹，文件名按顺序编号，例如：106-文章标题.md
2. 双击网站文件夹里的「更新网站.bat」，点「开始更新」
3. 用 GitHub Desktop 提交（Commit → Push origin），约 1 分钟后网站更新

提示：
- 第一行「# 标题」和第二行「> 分类 | 日期」必须保留，分类可填：诗 / 词 / 散文 / 其它
- 本模板只是参照，不要放进 articles 文件夹
